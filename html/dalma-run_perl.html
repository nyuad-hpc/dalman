Dalma Man Pages (dalman)

SYNOPSIS

NYUAD HPC team provides a global Perl software environment which can be used through "Linux environment modules". We do not install additional Perl modules to the global environment. So if you need, say the Test::More Perl module, you will need to "localise" the global Perl environment to your home directory and install the Test::More module yourself. Here are the instructions:

NOTE: The localisation is based on Perl's local::lib

DESCRIPTION

1.- Login Dalma.

2.- Run this in terminal to load Perl module:

----------------------------------------------------------

module load perl

----------------------------------------------------------

3.- At this point, perl is available for use. If you want to install localized packages, follow the steps below.

4.- (One time setup only) Run cpan to config. Run this in terminal

----------------------------------------------------------

cpan

----------------------------------------------------------

Say yes when it try to automate the process:

----------------------------------------------------------

Would you like to configure as much as possible automatically? [yes]

----------------------------------------------------------

And use local::lib:

----------------------------------------------------------

What approach do you want? (Choose 'local::lib', 'sudo' or 'manual') [local::lib]

----------------------------------------------------------

In the end, it is easier to save the localized commands to your bashrc. Then everytime you login, the localization is done. Example terminal output:

----------------------------------------------------------

Would you like me to append that to /home/gh50/.bashrc now? [yes]

----------------------------------------------------------

Finally, fix BeeGFS for CPAN: In file '~/.cpan/CPAN/MyConfig.pm', change

----------------------------------------------------------

'build_dir' => q[/home/YOUR-NET-ID/.cpan/build]

----------------------------------------------------------

to

----------------------------------------------------------

'build_dir' => q[/scratch/YOUR-NET-ID/.cpan/build]

----------------------------------------------------------

Now cpan should be working.

4.- (One time only setup only) Now go ahead and install cpanm, run this in terminal: ----------------------------------------------------------

cpan App::cpanminus

----------------------------------------------------------

Then run this in terminal, or put it in your ~/.bashrc and re-login:

----------------------------------------------------------

export PERL_CPANM_HOME=$SCRATCH/.cpanm

----------------------------------------------------------

5.- Now you could use cpanm to install your Perl package. Example installing "Test::More":

----------------------------------------------------------

cpanm Test::More

----------------------------------------------------------

Example screen output:

  [gh50@login-0-1 ~]$ cpanm Test::More
